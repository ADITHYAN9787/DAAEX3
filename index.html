Implementation of Kruskal's and Prim's Algorithms

Minimum Spanning Tree (MST) - Design and Analysis of Algorithms Lab

📖 Problem Statement

Given a weighted undirected graph containing 7 vertices (0–6) and 9 edges, implement both Kruskal's Algorithm and Prim's Algorithm to compute the Minimum Spanning Tree (MST).


The project verifies that both algorithms produce the same minimum total weight and compares their time complexity.

🎯 Objectives

  • Implement Kruskal's Algorithm.
  • Implement Prim's Algorithm.
  • Generate the Minimum Spanning Tree.
  • Compare the total MST weight.
  • Analyze algorithm complexity.

⚙️ Algorithms Used

Algorithm Approach Data Structure
Kruskal's Greedy (Edge Based) Disjoint Set (Union-Find)
Prim's Greedy (Vertex Based) Priority Queue (Min Heap)

⏱️ Time Complexity

Algorithm Time Complexity Space Complexity
Kruskal's O(E log E) O(V)
Prim's (Heap) O(E log V) O(V)
